# StockBuddy

StockBuddy is an educational stock analysis platform designed to help retail investors understand financial metrics and valuations. 

## Features
- **Live Stock Data**: Search and track real-time stock prices and trends.
- **Educational Tooltips**: Explain Like I'm 5 (ELI5) definitions for complex financial terminology everywhere.
- **Enhanced DCF Calculator**: Professional-grade Discounted Cash Flow valuation simplified for everyday investors.
- **Peer Comparison**: Compare stocks across fundamental metrics.
- **Financial Health**: Quick snapshot of a company's financial stability.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, shadcn/ui, Recharts
- Backend: Python, FastAPI (Hosted on Render)
- Deployment: Vercel (Frontend), Render (Backend)

## Running Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend and backend simultaneously:
   ```bash
   npm run dev
   ```

The frontend will be available at http://localhost:8080 and the backend at http://localhost:8000.
